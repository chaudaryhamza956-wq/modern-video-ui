#modern-video-ui
A responsive video landing page built with HTML and CSS. Features include a modern UI, glassmorphism design, and a content grid layout optimized for all screen sizes.

## ✨ Features
*   **Modern Premium UI:** Built with sleek spacing and a dark aesthetic.
*   **Glassmorphism Effects:** Uses modern CSS features like `backdrop-filter` for a premium glass feel.
*   **Responsive Content Grid:** Completely optimized using CSS Grid and Flexbox for mobile, tablet, and desktop screens.
*   **Accessible Form Elements:** Semantic HTML5 structure for better usability.

## 🛠️ Technologies Used
*   **HTML5:** Structured layout using semantic tags.
*   **CSS3:** Custom properties, CSS Grid, Flexbox, Media Queries, and Glassmorphism.
